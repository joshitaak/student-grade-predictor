# Student Grade Predictor

## 🚀 Overview
A machine learning model that predicts student performance based on input features like study time and attendance.

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas

## ⚡ Features
- Predicts grades using regression
- Data preprocessing and model training
- Accuracy evaluation

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
